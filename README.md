# Medical Insurance Premium Prediction

## Overview
A machine learning project that predicts medical insurance premiums based on patient data. The project covers the full data science workflow — from exploratory analysis to a deployed web application.

## Project Structure
- **1-eda/** — Exploratory Data Analysis (EDA) of the insurance dataset
- **2-advanced-analysis/** — Advanced modelling, evaluation, and a Flask web app for live predictions

## Tools & Technologies
- Python, Pandas, NumPy
- Matplotlib, Seaborn (visualisation)
- Scikit-learn (machine learning)
- Flask (web application)

## How to Run the Web App
1. Navigate to `2-advanced-analysis/webapp/`
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python app.py`
4. Open your browser at `http://localhost:5000`
